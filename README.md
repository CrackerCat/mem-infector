# mem-infector

## How to build the examples:

```shell
  mkdir build
  cd build
  cmake ..
  make
```

## How to use:
```
  usage: ./src/virus/virus --pid=int [options] ... 
options:
  -p, --pid      set target pid (int)
  -l, --libso    set libso name (string [=])
  -?, --help     print this message
```
